> 存放AISoul推广用的所有文案、图片和发布模板
> 品牌名：AI徕风灵魂实验室

## 目录结构

```
aisoul-content/
├── README.md              ← 本文件
├── CONTENT-GUIDE.md       ← 文案生成规范
├── CONTENT-RESEARCH.md    ← 小红书内容调研报告
├── template/              ← 内容模板
├── images/                ← 通用配图
└── 001-005/              ← 选题文件夹
```

## 快速使用

1. 打开对应选题文件夹
2. 复制 `content.md` 内容到小红书
3. 复制 `hashtags.txt` 标签
4. 根据 `cover-suggestion.md` 做封面图
5. 发布
