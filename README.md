# AISoul 小红书内容库

## 目录结构

每个选题一个文件夹，格式：序号-标题

每个文件夹包含：
- content.md — 可直接复制的小红书正文
- hashtags.txt — 推荐标签
- cover.md — 封面图片文案建议
- images/ — 配图存放

## 使用流程

1. 找到对应选题文件夹
2. 复制 content.md 内容到小红书
3. 从 hashtags.txt 复制标签
4. 配图后发布
