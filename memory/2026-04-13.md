### Weak项目大改版（人性弱点测试）

**背景**：Typhoon 反馈结果页显示失败，要求全面修复和优化。

**问题诊断**：
- 旧量表题计分：`scaleScores = [0,1,2,3,2,1,0]`（中间最高=最弱点，两端最低=无弱点）—— 逻辑反了
- 雷达图归一化：所有维度用同一 `/8` 系数，但各维度题数不同，上限不一致
- 后端 `calculate_scores` 对 Q20-Q25（旧格式5选项）计算报错，导致结果页500

**修复方案**：
- 前端完全接管计分逻辑，后端只传原始答案
- 量表题改为7点Likert（0-6线性分），题干给出"弱点观点"（越符合=越弱点）
- 各维度独立归一化上限（批评14分/冷漠14分/自恋12分/防御12分/索取12分/记不住10分）
- 雷达图用 `dims[d] / dimMax[d] * r` 绘制，保证各维度可比

**题目结构调整（Q1-Q25）**：
- Q1-Q19：5选项MC题，场景+题干分开，间隔加大
- Q20-Q25：7点量表题，场景+弱点陈述，完全符合=6分，完全不符合=0分

**六维度分配**：
- 批评型（Q1,5,9,13,20）：MC题4道 + 量表1道
- 冷漠型（Q3,7,10,11,21）：MC题4道 + 量表1道
- 自恋型（Q14,15,19,22）：MC题3道 + 量表1道
- 记不住型（Q16,17,23）：MC题2道 + 量表1道
- 防御型（Q2,6,12,24）：MC题3道 + 量表1道
- 索取型（Q4,8,18,25）：MC题3道 + 量表1道

**MC计分**（-1/0/0/1/2）：
- A=-1（最健康）, B=0, C=0, D=1, E=2（最弱点倾向）
- E的人可能是不作为/回避，不一定直接等于某弱点

**量表计分**（0-6线性）：
- 位置0（完全不符合）=0分
- 位置6（完全符合）=6分
- 总分 = MC得分(已转为非负) + 量表得分

**维度鉴定阈值**：
- 得分 ≥ 该维度满分的60% → 核心弱点
- 得分 30%-60% → 次要弱点
- 得分 < 30% → 安全区

**人际健康指数**：
- TOTAL_MAX = 70分（所有维度最大之和）
- 健康指数 = 100 - (总分 / 70 × 100)
