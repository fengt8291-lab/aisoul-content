## 💳 全球金融解决方案

### 核心发现：不需要永居也能开外国银行账户

**Wise（最适合Typhoon的方案）**
- 中国公民凭护照可完全线上开户
- 获得美国银行账户（Routing + Account Number）
- 欧盟账户（IBAN）、英国账户
- 绑定Stripe/PayPal收款，50+货币
- 费用：0.5-2%手续费

**其他选项**
| 方案 | 难度 | 优点 | 缺点 |
|------|------|------|------|
| Wise | ⭐最简单 | 护照就能开，线上完成 | 非真正银行 |
| Mercury（美国）| ⭐⭐ | 真正商业银行账户 | 2025年审核变严 |
| HSBC香港 | ⭐⭐ | 强大功能 | 需要去香港 |

**实际卡点**
1. 个人外汇限额$50,000/年（公司账户可绕过）
2. Stripe不支持中国大陆居民（但支持绑定美国银行账户）

**低成本全球金融体系路径**
```
第一步：注册Wise（立即可做）
第二步：毕业后/收入稳定后注册香港公司
第三步：公司账户收款无外汇限额
```

| @levelsio (Pieter Levels) | Nomad List $250K+/月 | 荷兰，一个人做旅行社区 |

### 成功共性
1. **从解决自己问题开始** → 先有产品，后有市场
2. **Twitter Build in Public** → 积累粉丝变成客户
3. **都是B2C，不是B2B** → 个人买单，不需要销售团队
4. **从小价格起步** → $4-79/月，快速迭代

### B2C vs B2B（重要结论）
- B2C：销售难度低、适合一个人、靠内容营销
- B2B：需要销售团队、不适合独立开发者
