# AISoul 小红书内容文案规范

## 一、核心定位

**账号人设：** AI徕风灵魂实验室 — 用AI探索情感世界的年轻女性视角
**目标用户：** 18-30岁小红书女性，对AI好奇、对情感有需求
**内容调性：** 轻松、有趣、有点共鸣感，不说教，不学术

---

## 二、文案结构模板

### 2.1 普通帖（不引流）

```
【标题】
引发好奇/共鸣的疑问句，或直接利益点

【正文第一句】
直接切入场景，让用户有代入感

【正文主体】
讲故事/提问题/对比内容
节奏：短句为主，一句话一行
每3-4句一段

【互动引导】（最后）
一句话引发评论，用问号结尾
例：你们觉得呢？/ 你有没有类似的经历？

【标签】hashtags.txt
```

### 2.2 引流帖（每5篇最多1篇）

```
【标题】
同普通帖结构

【正文】
同普通帖结构

【引流句】
"... 想去试试的话，链接在评论区"

【品牌露出】（网址后换行）
AI徕风灵魂实验室

【标签】hashtags.txt
```

---

## 三、写作要求

### 3.1 标题规范
- 数字要具体：不是"几个AI"而是"5个AI"
- 用emoji点缀情绪：😱😭🤔😂
- 疑问句 > 陈述句
- 控制在20字以内

### 3.2 正文规范
- **开头**：第一句必须让用户觉得"跟我有关"
- **节奏**：短句为主，每句话不超过25字
- **代入感**：用"我/我们/你"代替"人们/用户"
- **禁止**：太长（超过300字）、太学术、太广告腔
- **推荐**：每篇一个核心情绪点（震惊/共鸣/好奇/好笑）

### 3.3 标签规范
- 每篇10-15个标签
- 结构：1个主标签 + 3个话题标签 + 3个情绪标签 + 3个长尾标签
- 热门标签：#AI #人工智能 #chatgpt #mbti #情感 #恋爱
- 禁止：政治、宗教、敏感话题

---

## 四、引流规则（重要！）

| 规则 | 说明 |
|------|------|
| 引流比例 | **每5篇最多1篇**带网址 |
| 品牌露出 | 带网址时，网址**下一行**加"AI徕风灵魂实验室" |
| 引流形式 | 文字引导"链接在评论区"，不直接贴完整URL |
| 禁止行为 | 不能每篇都引流、不能在正文中硬广 |

---

## 五、内容选题方向

### 5.1 高互动选题（优先做）
- AI对比类：同一问题不同AI的不同回答
- 自我分析类：用AI分析自己/朋友的聊天记录、头像、朋友圈
- 灵魂测试类：AI MBTI / 灵魂类型 / 依恋类型
- 情感共鸣类：恋爱困惑、分手复合、暧昧期

### 5.2 次优选题
- AI使用技巧：怎么用AI辅助处理感情问题
- 趣味测试：让AI帮你做决定（今天穿什么/吃什么）

### 5.3 禁止选题
- 政治、宗教话题
- 拉踩其他AI产品
- 虚假夸大的AI能力描述

## 六、质量自检清单

发布前检查：
- [ ] 标题是否在20字以内？
- [ ] 开头是否能让用户有代入感？
- [ ] 正文是否短句为主、节奏清晰？
- [ ] 是否超过300字？（超过则精简）
- [ ] 互动引导是否为问句？
- [ ] 标签是否在10-15个？
- [ ] 是否为引流帖？如是，检查品牌露出是否完整。

---

## 七、文件夹命名规范

格式：`序号-选题关键词`

示例：
- `001-AI-MBTI测试`
- `002-灵魂伴侣对比`
- `003-AI头像分析`

每个文件夹必须包含：
- `content.md` — 正文文案
- `hashtags.txt` — 标签
- `cover-suggestion.md` — 封面文案建议（可选）
- `images/` — 配图文件夹（可选）
